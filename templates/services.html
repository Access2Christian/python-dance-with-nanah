Technical Documentation

MIVA-CSC312 Submission

1. Executive Summary

This application, "Python Dance with Nanah," is a comprehensive web portal built to demonstrate proficiency in the Flask framework, secure authentication protocols, and relational database management. It satisfies all core requirements of the Tutor-Marked Assessment (TMA) by bridging professional UI design with backend security best practices.

2. Core Assignment Requirements

Backend Architecture

Initialized in App.py using Flask. Implements dynamic routing for multi-page rendering and session-based user authentication.

Data Security

Integrated Werkzeug.security to implement PBKDF2-SHA256 password hashing, ensuring zero-knowledge storage of user credentials.

Persistence Layer

Developed using SQLAlchemy ORM with a tbl_user schema. Chosen SQLite for zero-config portability for the grading process.

Frontend Logic

Responsive UI built with Bootstrap 5. Utilized Jinja2 inheritance (layout.html) to ensure modular and DRY (Don't Repeat Yourself) code.

4. Initialization Guide

Step 1: Environment Setup

pip install flask flask-sqlalchemy werkzeug

Step 2: Start Application

python App.py

The system will automatically generate miva_nexus.db and create tbl_user on the first run.
